A Python + SQLite Based Inventory Management System

The Pharmacy Inventory & Expiry Tracker is a simple yet powerful desktop-based application designed to help pharmacies, medical shops, and healthcare workers efficiently manage medicines, track stock levels, and monitor expiry dates. Built using Python, Tkinter, and SQLite, this system ensures smooth inventory management and timely expiry alerts.
